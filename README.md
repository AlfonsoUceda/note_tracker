# NoteTracker

This gem provides the feature to track your notes with pivotal tracker

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'note_tracker'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install note_tracker
